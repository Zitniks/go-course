# Настройка окружения и ИИ-помощники

## Вы узнаете:
- **Необходимое ПО:** какой редактор и инструменты нужны для старта разработки на Go.
- **Интеграция:** как настроить VS Code для работы с Go на macOS и Linux.
- **Использование ИИ:** как применять ChatGPT для разбора теории, отладки кода и подготовки к интервью.
- **Практика с ИИ:** решение алгоритмических задач, поиск Race Conditions и проектирование System Design.
