# Базовый Git

## Вы узнаете:
- **Концепция VCS:** зачем нужна система контроля версий и как она избавляет от путаницы с копиями папок.
- **Локальный цикл работы:** создание репозитория (`init`), подготовка файлов (`add`) и фиксация изменений (`commit`).
- **Навигация и история:** просмотр списка изменений (`log`) и перемещение между состояниями проекта (`checkout`).
- **Работа с ветками:** создание изолированных линий разработки (`branch`) и их слияние (`merge`).
- **GitHub и удалённая работа:** клонирование проектов (`clone`), отправка кода (`push`) и получение обновлений (`pull`).
- **GitHub UI:** создание репозиториев, оформление Pull Request и визуальное решение конфликтов.

Разбирается база: `git init`, `git add`, `git commit` и навигация по истории через `git checkout`. Вводится понятие веток — как отделиться от основной линии, сделать изменения и объединить их обратно через `merge`.
